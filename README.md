## ✨ Features

- 📱 **Fully Responsive** — works on all screen sizes
- 🎨 **Modern design** — smooth animations, clean layout
- 🚀 **Zero dependencies** — pure HTML/CSS/JS, no frameworks
- ⚡ **Blazing fast** — static files served via GitHub Pages
- 📬 **Contact section** — links to GitHub, LinkedIn, freelance platforms
- 🗂️ **Projects showcase** — highlights key work with live/repo links

## 🚀 Getting Started

### Run Locally

```bash
# Clone the repository
git clone https://github.com/Tirumala135/myportfolio.git
cd myportfolio

# Open in browser (no build step needed)
open index.html
# or use VS Code Live Server extension
```

### Deploy to GitHub Pages

This repo is already configured for GitHub Pages. Push changes to `main` and the live site updates automatically.
